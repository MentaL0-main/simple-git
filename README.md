# Simple git

This pack of scripts, created for simple use git.

## Scripts
```python
ga - git add
gc - git commit [comment]
gp - git push
gacp - ga + gc + gp. use: gacp [commit comment]
gl - git log
gf - git fetch
gpl - git pull
gs - git status
gm - git merge
gchc - git checkout
```

## Quick start

For linux you can:

```bash
# Clone repo
git clone https://github.com/MentaL0-main/simple-git.git
cd simple-git

# Make to executable
chmod +x ./scripts/**

# Copy in /usr/bin for use anywhere
sudo cp ./scripts/* /usr/bin
```

Reboot terminal and use
